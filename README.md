# Power BI Audit & Performance Dashboard

## Overview
This project is a Power BI–based analytics solution designed to monitor audit quality, employee performance, workload distribution, and defect trends across departments. It provides management-level visibility into operational efficiency and quality risks.

## Business Problem
Audit and operational data was scattered across multiple files, making performance tracking and decision-making inefficient. This dashboard centralizes all audit-related data into a single analytical model.

## Dashboards Included
- Quality Performance Nexus – Audit quality scores and defect trends
- Departmental Drill Down – Department-wise performance and workload
- Workload & Defect Insights – Relationship between workload and defects
- Employee Insights Arena – Individual performance trends
- Auditor Analytics Panel – Auditor benchmarking and KPIs

## Key KPIs
- Audit Quality Score
- Defects per Audit
- Workload Utilization %
- SLA Compliance %
- Auditor Productivity Index

## Tools Used
- Power BI Desktop
- DAX
- Power Query
- Excel as data source

## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Update data source paths if required
4. Refresh the report
